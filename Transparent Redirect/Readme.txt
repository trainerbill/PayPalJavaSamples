Using Sample Code:
----------
Download paypalfunctions.jsp and transparentRedirect.jsp into your Java application
Open paypalfunctions.jsp and update your Payflow credentials:
	gv_APIUser	= "";//Enter USER field
	gv_APIPassword	= "";//Enter PWD field
	gv_APIVendor = "";//Enter VENDOR field
	gv_APIPartner = "PayPal";
Run transparentRedirect.jsp
